- 2018.05
  - 老齢基礎年金の受給条件
    - 10年以上の受給資格期間を満たした人が厚生年金保険に加入していた期間がある場合65歳から受給できる
  - 定期保険特約
    - 終身保険をベースにしつつ、一定期間の保険を厚くする。保険料を終身保険よりは割安
    - 更新時に告知は不要
  - 医療保険適用の条件
    - 1入院あたりの支払い限度日数は決まってる
    - 退院翌日から180日以内に同じ病気で入院した場合、1入院とみなされる
  - 損害保険、純保険料、付加保険料
    - 給付・反対給付均等の法則
    - 利得禁止の原則
    - 純保険料は保険金支払いのために使われるお金
    - 付加保険料は保険会社の経費などに使われるお金
  - 退職手当に関する税金(P139)
    - 退職所得 = (収入金額 - 退職所得控除額) x 1 / 2
    - 退職所得控除額
      - 20年以下 40万円 x 勤続年数
      - 20年以上 800万円 + 70万円 x (勤続年数 - 20年)
    - 別個課税
  - 不動産登記の記載(P161)
    - 表題部 土地や建物の表示に関する事項
    - 権利部 甲 所有権に関する事項 乙 所有権以外の権利に関する事項(抵当権)
  - 借地借家法の内容(P166)
    - 普通借地権 存続期間30年以上 契約更新可能 貸主が拒否するには正当事由が必要。期間の定めは30年以上（更新ごとに20年以上、10年以上）定めがない場合は、30年、20年、10年とみなす
    - 定期借地権 契約更新がなく返却が前提 一定定期借地権、建物譲渡特約付借地権、事業用定期借地権等の3種類がある
      - 一定定期借地権
      - 建物譲渡特約付借地権
      - 事業用定期借地権等
    - 普通借家権 契約更新できる 期間の定めはないが1年未満の契約は期間の定めがないものとする 貸主が拒否するには正当事由が必要
    - 定期借家権 契約更新がなく返却が前提 1年未満の契約も可能
  - 相続割合のレアケース(P194)
    - 基本は配偶者
    - 配偶者が死亡などでいないケースはその分が第一、第二、第三と渡っていくイメージ
  - 保険の任意継続(P46)
    - 2ヶ月以上被保険であった
    - 20日以内に申請
    - 最長2年間
  - 教育ローンの内容(P40)
    - こども保険 保険料を支払い、満期時に保険金を受け取れる。親が死亡した場合、保険料は免除され保険金は支払われる
    - 教育ローン 350万円(要件次第で450万円)/1人。最長15年
    - 奨学金 利息なしの第一種と利息ありの第二種
  - 生命保険の延長保険など対応(P76)
    - 保険料が重い場合、以下の方法を利用できる
      - 延長保険 解約払戻金を利用して、保険金と同額の支払いを行う。保証内容は維持し、満期を短くするイメージ
      - 払済保険 保険料の払い込みを停止し、解約払戻金で支払える内容に変更する。保証内容を薄くし、満期を維持するイメージ
  - 個人年金保険(P82)
    - 変額 保険料を特別勘定(他の保険資産と分けられる)として運用する
  - 小売業者などの事業に関わる保険(P95)
    - 資産などではなく、利益に備えるのは、企業費用・利益総合保険
  - 経済の金利の動き(P100)
    - P100をさらっと読み直す
  - 譲渡にかかる税金(P182)
    - 土地・建物の譲渡は、譲渡した日の属する年の1月1日までの所有期間が5年を超えるかどうか。短期と長期で判断
    - 短期譲渡 39%(所得税30%、住民税9%)
    - 長期譲渡 20%(所得税15%、住民税5%)
  - 住宅ローン・住宅借入金等特別控除の条件(P148)
    - 取得日から6ヶ月以内に入居
    - 所得が3,000万円以下
    - 50m2以上で1/2以上が居住用
    - 返済期間が10年以上
  - 区分所有(P168)
    - 集会の召集
      - 1/5以上
    - 普通決議事項
      - 過半数 管理者の専任・解任、大規模修繕など
    - 特別決議
      - 3/4以上 規約の設定、変更、廃止など
      - 4/5以上 建て替え
  - 贈与契約(P208)
    - 書面 贈与契約の効力が発生した日
    - 口頭 贈与の履行があった日
  - 相続税の非課税限度額(P200)
    - 死亡保険金・死亡退職金の場合 500万円 x 法定相続人の数(放棄したものを含む)
    - 遺産相続の場合 600万円 x 法定相続人の数(放棄したものを含む)
    - 民法の場合、放棄したものは、初めから相続人でなかったとみなされる
    - 相続税の計算の場合、放棄はなかったものとして計算される
  - 特定居住用宅地などの相続税と評価額(P216)
    - 330m2まで80%の減額
    - 路線価(P215)
    - 建築基準法(P172)
  - マンションの減価償却
  - 居住用不動産の贈与(P210)
    - 贈与税の配偶者控除 2,000万円
    - 贈与税の基礎控除 110万円
  - 老齢基礎年金の繰り上げ(P61)
    - 繰り上げの場合、0.5%/月の減額
    - 繰り下げの場合、0.7%/月の増額
  - 付加年金(P61)
    - 国民年金第一号被保険者が月額400円上乗せ支払いすると、老齢年金の受給額が200円 x 付加保険料納付月数上乗せされる
